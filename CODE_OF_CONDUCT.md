# Code of Conduct

Mind Your Skill aims to be a welcoming open-source project.

## Expected behavior

- Be respectful.
- Give constructive feedback.
- Keep discussions focused on improving skills and documentation.
- Respect user privacy and safety.

## Unacceptable behavior

- Harassment or personal attacks
- Publishing private data or secrets
- Submitting malicious skill behavior
- Intentionally hiding risky actions inside skill instructions

## Enforcement

Project maintainers may remove content, close issues, or reject contributions that violate this code of conduct.
